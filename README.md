# Kraken LTP 

## Usage

```bash
  // run application using docker-compose
  make run
```

```bash
    // run tests
    make tests
```

```bash
    // run swagger to generate swagger.yaml file
    make swagger
```

```bash
    // run format-code to reformat code
    make format-code
```

## API
```
    GET /api/v1/ltp (for more details see docs/swagger.yaml)
```
